# Analytical Questions I Can Answer

This document lists the main questions that can guide the analysis of the 2022–2023 football player statistics dataset.

## Main Questions

* Which players were the most efficient based on per-90 metrics?
* How does performance vary across positions?
* Which players stand out as finishers, creators, defenders, and progressors?
* Which players create value beyond goals and assists?
* Which players may be underrated because they performed well despite fewer minutes?
* Can players be grouped into meaningful performance profiles using clustering?

## Offensive Performance

* Which players scored the most goals?
* Which players had the highest goals per 90 minutes?
* Which players had the highest goal contribution?
* Which players were the most efficient finishers?
* Which players had high shot volume but low scoring efficiency?

## Passing Performance

* Which players completed the most passes?
* Which players had the highest pass completion percentage?
* Which players contributed most to ball progression through passing?
* Which players completed the most passes into the final third or penalty area?
* How does passing style vary by position?

## Chance Creation

* Which players created the most shot-creating actions?
* Which players created the most goal-creating actions?
* Which players created value without necessarily scoring?
* Which players are stronger creators than finishers?

## Defensive Contribution

* Which players contributed most through tackles, interceptions, blocks, and clearances?
* Which players were most active defensively?
* Which players made the most errors leading to opponent shots?
* How does defensive contribution vary by position?

## Possession and Ball Progression

* Which players had the most touches?
* Which players carried the ball the most?
* Which players progressed the ball most through carries?
* Which players received the most progressive passes?
* Which players were most involved in attacking areas?

## Dribbling and Take-Ons

* Which players attempted the most take-ons?
* Which players completed the most successful take-ons?
* Which players had the highest take-on success rate?
* Which players were strongest in one-vs-one situations?

## Discipline and Fouls

* Which players committed the most fouls?
* Which players drew the most fouls?
* Which players received the most cards?
* Which players had the highest disciplinary risk?

## Aerial Duels

* Which players won the most aerial duels?
* Which players had the highest aerial duel success rate?
* Which positions are strongest in aerial duels?

## Position Analysis

* How does performance vary by position?
* Which metrics are most relevant for each position?
* Should players be compared globally or within their position group?

## Squad and Competition Comparison

* How do players from different squads compare?
* How do players from different competitions compare?
* Which squads have more offensive, defensive, or progressive profiles?

## Player Profiles and Clustering

* Can players be grouped into meaningful profiles?
* Which players have similar statistical profiles?
* Can we identify finishers, creators, defenders, progressors, and balanced players?
* Do the clusters align with player positions?

## Dataset Limitations

This dataset does not answer match-by-match performance, player salary, transfer value, injuries, tactical role, opponent quality, or tracking-based questions.
